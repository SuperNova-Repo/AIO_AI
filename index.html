Puter.js Authentication

Authentifizierung mit Puter AI Chat API

Auth-Workflow:

  • Puter.js wird als <script src="https://js.puter.com/v2/"> geladen
  • Auth-Trigger: puter.ai.chat('test', {model:'claude-sonnet-4-5'})
  • Popup öffnet sich zu: puter.com?embedded_in_popup=true&request_auth=true
  • Erfolgs-Bedingung: res && res.success !== false
  • Nach Auth sind alle Puter-APIs verfügbar
